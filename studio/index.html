Hourly Packages and Rates

Web Studio projects are billed by the working hour. Working hours are pre purchased in bulk packages based on a range estimate developed for each project. Hourly rates for each package are determined by a standard formula based on the number of hours being purchased and which (if any) rate adjustments should be applied.
